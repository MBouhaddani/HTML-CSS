Onderzoek

Waarom noem je de eerste HTML-pagina; index.html?

De eerste HTML-pagina wordt "index.html" genoemd omdat webbrowsers deze naam standaard gebruiken voor de startpagina van een website.

Met welke HTML versie werken we nu?

We werken momenteel met HTML 5

Met welke CSS versie werken we nu?

We werken momenteel met W3.CSS 4

Hoe kan het dan een website er net iets anders uit ziet in een andere browser? (Edge, Chrome, Firefox, Opera)

Verschillende browsers interpreteren of tonen websitebroncode zoals HTML en CSS op iets verschillende manieren, wat resulteert in dezelfde website die er anders uitziet en aanvoelt.

Welke browser is het populairst en hoeveel procent RAM gebruikt deze browser?

Google Chrome is momenteel de meest gebruikte browser ter wereld. Het staat echter bekend om zijn hoge RAM-gebruik. Volgens tests kost het hebben van 10 tabbladen open uw RAM net iets minder dan 1000 MB.
